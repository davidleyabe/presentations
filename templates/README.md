# Templates

## Template padrão premium

Arquivo base:
- `templates/premium-landing-page.html`

Objetivo:
- servir como ponto de partida para novas apresentações
- manter o novo padrão visual premium em fundo branco com Tailwind CSS
- acelerar produção sem voltar ao layout antigo e pobre

## Como usar

1. Escolha o slug da apresentação.
2. Copie a estrutura do template para `presentations/<slug>/index.html`.
3. Substitua os placeholders `{{...}}` pelo conteúdo real.
4. Remova seções que não fizerem sentido.
5. Duplique blocos quando precisar de mais cards, timeline, cenários ou fontes.
6. Valide localmente no navegador antes de publicar.

## O que esse template garante

- hero forte com composição editorial
- bloco de executive snapshot
- ritmo visual melhor entre seções
- cards premium com sombras sutis
- linha do tempo mais sofisticada
- seção dark para monitoramento ou takeaway
- tipografia e espaçamento mais fortes que o padrão antigo

## Regras de design

- manter fundo branco como base
- usar Tailwind CSS
- priorizar hierarquia tipográfica forte
- evitar blocos repetitivos demais
- usar cores de acento com moderação
- preferir linguagem visual de briefing executivo premium
- não usar carrossel, formato de slides ou visual de template genérico

## Observação

O template é base, não prisão. A estrutura pode ser adaptada por tema, desde que preserve a direção visual premium definida para o repositório.
